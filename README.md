# Customer Shopping Trends EDA

## Overview

This project analyzes a customer shopping dataset to understand buying patterns, spending behavior, and subscription trends.

---

## Dataset

The dataset includes:

* Age
* Gender
* Category
* Purchase Amount (USD)
* Payment Method
* Subscription Status
* Season

---

## What I did

* Cleaned the dataset
* Explored distributions of key variables
* Compared spending across different groups
* Analyzed subscription behavior
* Created visualizations to understand patterns

---

## Key Findings

* Male customers have higher subscription rates than females
* Some product categories generate more revenue than others
* Spending varies across age groups
* Seasonal trends affect purchasing behavior


## Tools

* Python
* Pandas
* Matplotlib
* Jupyter Notebook
